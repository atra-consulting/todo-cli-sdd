```typescript
#!/usr/bin/env node

import { Command } from 'commander';

const program = new Command();

program
  .name('todo')
  .description('Manage todos in a local todos.json file')
  .version('1.0.0');

// Commands will be registered here in Phase 2

program.parse(process.argv);
```

Key requirements:
- Shebang `#!/usr/bin/env node` MUST be the first line — without it the bin field fails when invoked directly (Pitfall 10)
- Import uses `'commander'` (not a relative path) — works because commander is in node_modules
- `program.parse(process.argv)` at the end — synchronous I/O only, no async needed (all storage will be sync per D-12)
